# Professional Cloud Security Operations Engineer

This certification is for security professionals who focus on threat detection, incident response, and continuous monitoring within a Google Cloud environment.

## Exam Details
- **Level:** Professional
- **Length:** 120 minutes
- **Format:** Multiple choice and multiple select
- **Cost:** $200 USD

## Key Topics
- **Threat Detection:** Utilizing Security Command Center (SCC) and Chronicle.
- **Log Analysis:** Cloud Logging, BigQuery for log analysis, and log sinks.
- **Incident Response:** Playbooks, automation with Workflows, and forensic analysis.
- **Security Posture Management:** Identifying misconfigurations and compliance drift.
- **Network Security:** VPC Service Controls, Cloud Armor, and Firewall Insights.
