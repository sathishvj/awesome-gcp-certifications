# Associate Data Practitioner

This certification validates foundational skills in data ingestion, transformation, and analysis on Google Cloud. It is the entry point for those working with data pipelines and analytics.

## Exam Details
- **Level:** Associate
- **Length:** 120 minutes
- **Format:** Multiple choice and multiple select
- **Cost:** $125 USD

## Key Topics
- **Data Ingestion:** Using Pub/Sub, Dataflow, and Transfer Service.
- **Data Storage:** Understanding BigQuery, Cloud Storage, and Cloud SQL basics.
- **Data Transformation:** Basic SQL, Dataflow pipelines, and Dataprep.
- **Data Analysis & Visualization:** Looker Studio and basic BigQuery analytics.
- **Security & Governance:** IAM basics and data encryption at rest/transit.
